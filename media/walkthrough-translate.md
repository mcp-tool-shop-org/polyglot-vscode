## Translate Text

Three ways to translate selected text:

1. **Keyboard shortcut**: Select text, then `Ctrl+Alt+T` (`Cmd+Alt+T` on Mac)
2. **Right-click menu**: Select text → right-click → **Translate Selection**
3. **Command Palette**: `Ctrl+Shift+P` → **Polyglot: Translate Selection**

Pick a target language from the list (55 available) and the translation replaces your selection in-place.

### Supported languages

Arabic, Bengali, Chinese, Dutch, French, German, Greek, Hindi, Italian, Japanese, Korean, Portuguese, Russian, Spanish, Thai, Turkish, Vietnamese, and 38 more.
