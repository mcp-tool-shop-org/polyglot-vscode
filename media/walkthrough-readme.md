## Translate READMEs

The **Translate README** command is purpose-built for open source projects:

- Preserves code blocks, HTML badges, and table structure
- Translates headings, paragraphs, and table content intelligently
- Skips package names, URLs, and inline code
- Creates `README.ja.md`, `README.es.md`, etc. alongside your README

### Default languages

Japanese, Chinese, Spanish, French, Hindi, Italian, and Brazilian Portuguese.
